P0420 REPAIR RECOMMENDATIONS
============================================================
STEP 1: VERIFY THE CODE
- Clear the code and drive 20-30 miles
- If it returns immediately = likely bad cat
- If it takes several drive cycles = possible other causes

STEP 2: CHECK THESE FIRST (cheaper fixes):
- Downstream O2 sensor (Bank 1, Sensor 2)
- Exhaust leaks between engine and cat
- Air filter (dirty filter can affect mixture)
- Fuel injector cleaning

STEP 3: ADVANCED DIAGNOSTICS:
- O2 sensor switching test with scanner
- Catalytic converter efficiency test
- Check for oil consumption (blue smoke)

STEP 4: IF CAT REPLACEMENT NEEDED:
- OEM Honda cat: $800-1200 (best longevity)
- Aftermarket CARB compliant: $300-600
- Labor: 2-4 hours ($200-400)
